# todo

TODO app written in C++ (SOCI database library)

## Usage

```
todo [sub-command]
  list       : list todo items
  add [text] : add todo item
  done [id]  : done the todo
  undone [id]: undone the todo
  delete [id]: delete the todo
```

## Installation

```
$ mkdir build && cd build && cmake .. && make
```
